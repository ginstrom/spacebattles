# Space Battle

A space game written using pygame.

## Gameplay Demo

![Space Battle gameplay demo](assets/demo/gameplay.gif)

## Capture a New 10-Second GIF

```bash
./scripts/capture_demo_gif.sh --duration 10 --output assets/demo/gameplay.gif --fps 20 --display :0.0 --geometry 1280x720
```

Requirements:
- `ffmpeg` installed and on `PATH`
- An X11 display available at the selected `--display`
