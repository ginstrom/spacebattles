# Logging Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add Python `logging` to the game, starting with pause-state changes tagged with an in-game elapsed-time timestamp.

**Architecture:** A `game_time_ms` accumulator in `BattleScreen` advances only during active (unpaused, no-winner) frames. `main.py` configures logging once at startup to write to both `spacebattle.log` and stderr. `_toggle_pause()` emits one `INFO` line per toggle.

**Tech Stack:** Python `logging` (stdlib), pytest, uv

Add the log line at the end:

```python
def _toggle_pause(self, now):
    self.is_paused = not self.is_paused
    if self.is_paused:
        self.message = "Paused. Press Space to resume."
        self.cpu_fire_at_ms = None
    else:
        self.message = "Battle running. Press Space to pause."
        self.cpu_fire_at_ms = now + self.CPU_DELAY_MS
    _log.info("T+%.3fs %s", self.game_time_ms / 1000, "paused" if self.is_paused else "unpaused")
```
